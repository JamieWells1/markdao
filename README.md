# markdao

AI-powered learning worksheets, pure client-side application.

**Just go use it: [jamiewells1.github.io/markdao](https://jamiewells1.github.io/markdao/)**

---

You describe what you want to learn. You get a prompt. You paste it into any LLM. You paste back the worksheet. You fill it in. You copy the result back to get feedback. That's it.

Seriously, the site explains itself faster than this README can. Go try it.
